Channapatna Namma Pride 🎨🪵

**Overview**
Channapatna Namma Pride is an Android application developed to promote authentic Channapatna toys and support local artisans digitally. The app helps users explore toy details, verify authentic products, learn about the toy-making process, and access artisan and workshop information.

**Features**
Toy Verification using Toy ID
Digital Toy Catalog
Artisan Information
Workshop Location Details
User-Friendly Interface
Traditional Toy Promotion

**Technologies Used**
Android Studio
Kotlin
Jetpack Compose
Room Database
Google Maps API
Database Used

The project uses Room Database to store and manage toy, artisan, and workshop information locally within the application.

**Project Objectives**
Promote authentic Channapatna toys digitally
Support local craftsmen and artisans
Provide toy verification functionality
Increase awareness about traditional toy-making culture

**Methodology**
Data Collection
App Design and Development
Database Management
Testing and Implementation

**Results**
Successfully developed the Android application
Users can verify authentic toys
Provides toy, artisan, and workshop details
Helps promote traditional Channapatna toys digitally

**Future Scope**
Add online shopping and payment features
Provide multilingual support
Integrate AI-based recommendations

**Conclusion**
The Channapatna Namma Pride application helps preserve and promote Karnataka’s traditional Channapatna toys through modern Android technology. It provides a simple digital platform for users and supports local artisans effectively.
